A refreshing no-bake dessert featuring layers of silky matcha panna cotta, chewy mochi balls, and sweet red beans.

## Ingredients

| Ingredient | Amount |
|------------|--------|
| matcha powder |  |
| matcha powder | 6克 |
| gelatin |  |
| glutinous rice |  |
| warm water | 60克 |
| ⭐️辅料 |  |
| 蜜豆 |  |

## Seasonings

- whipping cream
- glutinous rice

## Instructions

1. 1、whipping cream➕纯milk➕白砂sugar混合均匀，用微波炉或者奶锅加热至微微烫手 ，倒出一点用来化开matcha powder粉 2、gelatin片提前用冷water泡软，放入热milk中搅拌融化，然后把matcha powder液倒入milk液中搅拌均匀，过筛一遍倒入模具，放入冰箱冷藏三个小时以上定型
2. 3、制作glutinous rice丸子：glutinous rice粉加sugar搅拌均匀 倒入warm water搅拌成絮状，再揉成团 ，搓成丸子 4、锅中放water，water开后放入丸子煮至浮起，捞出来过cold water备用
3. 5、然后把定型好的奶冻切块放入碗中➕蜜豆加➕丸子，倒入milk即可（我个人更喜欢加milk和椰浆一起😄）
4. 软糯小丸子➕香甜蜜豆➕茶香浓郁的奶冻，完美

## Tips

1、不管什么牌子的matcha powder粉都吸收湿气，形成结块。制作前建议过筛。若需与其他分类混合过筛的，事先混合均匀后再过筛会更好。 2、过筛后的matcha powder虽然细腻但是还是无法完全避免matcha powder颗粒的存在，想要达到完美的效果仙女可以考虑备一个茶筅哦😁😁
