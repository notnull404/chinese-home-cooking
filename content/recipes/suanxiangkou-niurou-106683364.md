Learn how to make Beef (suan-xiangkou-) with this easy recipe.

## Ingredients

| Ingredient | Amount |
|------------|--------|
| 牛肉 |  |
| 口菇 |  |

## Seasonings

- 大蒜
- 黄油
- 黑胡椒粉

## Instructions

1. 牛肉切小块，放入1勺生抽，半勺老抽，1勺耗油，适量黑胡椒粉抓匀，腌制30分钟
2. 口菇切片备用，大蒜切厚片备用
3. 把锅烧热，放入少许黄油，牛肉煎熟盛出备用
4. 另起锅烧油，倒入大蒜炒香，倒入口菇翻炒均匀，再倒入牛肉翻炒
5. 加入1勺生抽，1勺耗油，适量黑胡椒粉，翻炒均匀即可出锅

## Tips

For best results, use fresh ingredients.
