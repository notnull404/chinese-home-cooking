A hearty, healthy congee made with red beans, black beans, black rice, and milk — more satisfying than bubble tea.

## Ingredients

| Ingredient | Amount |
|------------|--------|
| red beans |  |
| 黑豆 |  |
| 黑米 |  |
| milk | 一 box |

## Seasonings

- sugar

## Instructions

1. 黑豆red beans浸泡一晚，记得放在阴凉处，这天气容易臭……
2. 黑米黑豆red beans洗净放入电饭煲中，加入四碗water，开始煲粥
3. 粥煮好后倒入milk，再焖五分钟咯，就奶香浓郁了，不减肥的给我使劲加sugar
4. 看看这卖相！主要好喝还养生

## Tips

For the best results, prepare all ingredients before you start cooking (mise en place).
