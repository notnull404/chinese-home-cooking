A classic Chinese summer beverage made from smoked plums, hawthorn, licorice and five other herbs, boiled into a sweet-tart refreshing tonic.

## Ingredients

| Ingredient | Amount |
|------------|--------|
| 乌梅 |  |
| 山楂 |  |
| 陈皮 |  |
| 甘草 | 3 slices |
| 桑葚 |  |
| 洛神花 |  |
| mint | 1克 |
| 桂花（可不加） | 3克 |
| water | 1.5升 |

## Seasonings

- rock sugar

## Instructions

1. 除桂花、rock sugar⚠️ 所有药材洗净后，加入1.5升water浸泡30分钟
2. 大火煮沸后转小火煮30分钟，出锅前按需放入rock sugar ⚠️出锅前可捞出/过滤食材 ⚠️最好用瓷瓦煲（实在没有可以考虑养生壶）
3. 出锅的时候撒桂花（已过滤药材）
4. 😋好看又好喝（没过滤药材）
5. 冰镇效果更加😍😍 可放入冰箱冷藏，可加入冰块🧊

## Tips

⚠️出锅前可捞出/过滤食材 ⚠️最好用瓷瓦煲（实在没有可以考虑养生壶）
